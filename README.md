# e-plantShopping

Paradise Nursery is a React + Redux shopping cart application for browsing indoor plants, adding them to a cart, and managing quantities before checkout.

## Features

- Landing page with a hero section and a `Comenzar` button
- About section with company details
- Product listing grouped by plant categories
- Shopping cart with increment, decrement, remove, and totals
- Dynamic cart count in the navigation bar

## Main Files

- `src/components/AboutUs.jsx`
- `src/App.css`
- `src/App.jsx`
- `src/features/cart/CartSlice.jsx`
- `src/components/ProductList.jsx`
- `src/components/CartItem.jsx`

## Run locally

```bash
npm install
npm run dev
```
